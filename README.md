# lofi-stream-rumble

24/7 lofi beats streamed to Rumble with a cozy library/study aesthetic.

## Live Stream

Watch: [Coming Soon on Rumble]

## Theme

Warm library study vibes with:
- Dark wood bookshelf filled with leather-bound books
- Comfortable reading nook with armchair
- Crackling fireplace
- Classical-inspired lofi with soft piano tones

## Local Development

```bash
cd docs
python3 -m http.server 8080
# Open http://localhost:8080
```

## Tech Stack

- HTML/CSS for visuals
- Web Audio API for generative classical-lofi audio
- GitHub Pages for static hosting
- Xvfb + Chromium + FFmpeg for streaming
- systemd for auto-recovery
